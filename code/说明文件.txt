【项目配置与运行】
1、配置本地Python环境 （选择自己的python编译器，建议创建一个虚拟环境）
2、安装依赖库，在终端窗口（Teminal）输入命令：
pip install -r requirements.txt -i https://pypi.mirrors.ustc.edu.cn/simple/
3、启动项目，在终端窗口输入命令：
python manage.py runserver
4、浏览器打开网站：
http://127.0.0.1:8000/
5、查看环境
conda info --envs
source activate myenv
6、数据库迁移
python manage.py makemigrations 自动生成数据库迁移脚本
python manage.py migrate 执行数据迁移
